# 🚀 Popup to Tab: Auto-Group & Tabify

[![Manifest V3](https://img.shields.io/badge/Manifest-V3-brightgreen.svg)](https://developer.chrome.com/docs/extensions/mv3/intro/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

> **Automatically convert cluttered popup windows into organized tabs and keep your workspace tidy with smart tab grouping.**

---

## ✨ Features

- **🔄 Auto-Tabification**: Transforms all new popup windows into new tabs in your primary browser window.
- **📁 Smart Grouping**: Seamlessly integrates new tabs into your existing Tab Groups.
- **⚡ One-Click Toggle**: Instantly enable or disable the functionality via the toolbar icon.
- **🎨 Modern Design**: Features a fancy Neon Mint icon with real-time ON/OFF status badges.
- **🛡️ Secure & Lightweight**: Built on the latest Manifest V3 for optimal performance and security.

### 🛠️ Manual Installation (Development)
1. Clone this repository:
   ```bash
   git clone https://github.com/your-username/edge-popup-to-tab.git
   ```
2. Open your browser's extension management page:
   - **Chrome**: `chrome://extensions`
   - **Edge**: `edge://extensions`
3. Enable **Developer mode** (top right).
4. Click **Load unpacked** and select the root directory of this project.
